Platform as a Service

Platform as a service is where a platform provider allows customers to access tools for application development over the internet. Usually, clients will use PaaS providers for services such as hosting or development, rather than using them as a substitute for an entire infrastructure. The main benefits to PaaS are the facts that the tools can be accessed from nearly any web browser, so it is not necessary for a company to install the software on all their systems, and the facts that PaaS is usually charged for on a per use basis rather than a flat fee, where the company may be paying a lot for a service they do not use much.


Cloud9


Cloud9 is an IDE hosted on the web that allows users to write and test code. It lets users code from any device with a website and allows them to continue their work anywhere without having to bring their development files on a hard drive with them, its all saved in the cloud. Cloud9 supports many languages such as PHP, Javascript and Python, and has built-in support for real time collaboration, making it the premier choice for companies seeking an IDE platform.
